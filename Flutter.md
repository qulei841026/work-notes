##安装
1.解压安装并配置环境变量<br>
```
    ./flutter/bin
```
2.开发二进制文件预下载
```
    flutter precache
```
3.设定镜像配置(环境变量里)
``` 
    PUB_HOSTED_URL=https://pub.flutter-io.cn
    FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```
4.安装依赖检测
```
    flutter doctor
```
